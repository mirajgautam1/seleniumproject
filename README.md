SETUP AND ENVIRONMENT:

2. Create and new project Directory
3. Ensure Java is installed on your system

TEST SCENARIOS:

1. Login with an Invalid User:
   
   Expected Result: You see an error message by logging in with an invalid user.
   Test Steps:
   
   i. Navigate to the login page.
   
   ii. Fill out the login form with invalid user details.
   
   iii. Submit the Login Form.
   
   Actual Result: Error Message is displaying with appropriate error message.

2. Login with a valid user:

   Expected Result: To verity that the user can be logged in with valid credentials.

   Test Steps:

   i. Navigate to the login page.

   ii. Fill out the login form with valid users.

   iii. Submit the login form.

   Actual Result:  Successfully verified that the user is logged in and redirected to the inventory page.

 3. Filter Products by Size and Price:

    Expected Result: To verify that a user can filter items by Name (descending order) and Price from Low to High.

    Test Steps:

    i. Navigate to the inventory page.
    
    ii. Apply a filter for Name (descending order).
    
    iii. Verify that the item listings are updated according to the applied filters.
    
    iv. Apply a filter for Price (Low to High).
    
    v. Verify that the item listings are updated according to the applied filters

    Actual Result: Successfully verified that the item listings are updated according to the applied filters.

4. Add items to the Cart:

   Expected Result: To verify that a user can add items to the cart.

   Test Steps:

   i. Navigate to the inventory page.
   
   ii. Hover over an item from the list and click the “Add to Cart” button. (Add two or more than two items.)

   Actual Result: Succesfully verified that the selected items are added to the cart.

5. Perform Checkout:

   Expected Result: To verify that a user can perform a checkout.

   Test Steps:

   i. Navigate to the cart page.
   
   ii. Proceed to the checkout process.
   
   iii. Fill in the necessary billing and shipping information.
   
   iv. Complete the checkout process.
   
   Actual Result: Succesfully verified that the checkout is successful and an order confirmation is received.
